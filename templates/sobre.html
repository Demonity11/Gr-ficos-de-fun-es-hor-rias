Sobre

O site Horárias foi construído usando como base o micro framework Flask em Python para o site e a biblioteca Chart.js de JavaScript para geração dos gráficos das funções.

Como funciona?

Na página de funções, nós temos 3 funções hórarias, sendo essas a posição no MU (Movimento Uniforme), velocidade e espaço no MUV (Movimento Uniformemente Variado).
Abaixo temos as 3 equações respectivamente:

\(S(t) = S₀ + v.t\)
\(v(t) = v₀ + a.t\)
\(S(t) = S₀ + v₀.t + \frac{a}{2} .t²\)
